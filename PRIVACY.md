# Privacy Policy — px

**px collects no data.**

This plugin consists entirely of static markdown files — skills, slash commands, and an agent definition. It contains no executable code, no hooks, no MCP servers, and makes no network requests.

- It does not collect, store, or transmit any personal information, usage data, or telemetry.
- It does not read or send your code anywhere. All processing happens inside your own Claude Code session, governed by [Anthropic's privacy policy](https://www.anthropic.com/legal/privacy).
- The plugin has no server-side component and no third-party services.

Questions: open an issue at [github.com/MohammadShehadeh/px](https://github.com/MohammadShehadeh/px/issues).
